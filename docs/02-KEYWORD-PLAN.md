## How the Admin Keyword Engine uses this

The seed file at `content/keywords/seed.json` is loaded into the admin **Keyword Planner**. From there each keyword gets:
- Status (`idea` → `writing` → `published` → `update`)
- Suggested article title + outline
- Cluster + funnel stage
- Linked competitor URLs
- Linked monetization (affiliate program ID)
- Auto-generated post draft via the bulk generator script
